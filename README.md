# EkiFlow - 駅旅の記録アプリ

## 概要

EkiFlowは、日本全国の鉄道駅を巡る旅を記録するiOSアプリです。訪れた駅を記録し、あなただけの「駅コレクション」を作りましょう。

## 主な機能

### 🚉 駅の記録
- 全国9,000以上の駅に対応
- 訪問日時・写真・メモを記録
- 「🌸行った」「🌷乗換」「🌿通過」「🏠最寄り」のステータス管理

### 🗺️ ルート検索
- 出発駅と到着駅を選ぶだけで経路を自動計算
- 快速線と各停の区別に対応
- 徒歩乗り換え（水天宮前⇔人形町など）も考慮
- 複数の経路候補から選択可能

### 🚃 路線一括登録
- 路線を選んで複数駅を一括で記録
- ログタブでグループ化して表示

### 📊 統計・コレクション
- 都道府県別の訪問率
- 路線別の制覇率
- 訪問駅数の推移

### 🗾 地図表示
- 訪問した駅を地図上に表示
- 現在地周辺の駅を確認

## デザイン

桜をモチーフにした和風テーマで、駅を巡る旅を彩ります。

## プロジェクト構成

```
EkiFlow/
├── Models/          # データモデル
├── Views/           # UI画面
├── ViewModels/      # ビジネスロジック
├── Services/        # ルート検索等
└── Resources/       # 駅データJSON
```

## セットアップ

```bash
# クローン
git clone https://github.com/tjx4ut/EkiFlow.git

# Xcodeで開く
cd EkiFlow
open EkiFlow.xcodeproj

# ビルド & 実行 (Cmd + R)
```
